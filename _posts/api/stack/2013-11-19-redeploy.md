Starts the redeployment of a stack on user's behalf.

### Request

* None

### Response

#### Success

<code class="inline-code">{ "ok" : true,  "message" : "Stack queued for redeployment" }</code>

#### Error

```bad_request - no stack UID provided
not_found - invalid stack UID provided```
