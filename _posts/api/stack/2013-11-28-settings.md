Gets a list of all applicable stack settings.

### Request

* None

### Response

A typical response will look like this:

<code class="inline-code">{
	"git.branch" : "master",
	"git.repository" : "https://github.com/cloud66/help.git",
	"reconfigure.nginx" : "false"
}</code>
